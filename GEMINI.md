# VibeVault Project Summary

A personal "Mood Board" application to save visual memories as digital Polaroids.

## Tech Stack

- Next.js
- Tailwind CSS
- Framer Motion
- Lucide React

## Project Commands

- **Development:** `npm run dev`
- **Build:** `npm run build`
- **Start:** `npm run start`
- **Lint:** `npm run lint`
